# wb_parser

Парсер каталога товаров Wildberries по поисковому запросу. Собирает данные с API и CDN и сохраняет в XLSX.

## Установка

```bash
poetry install
```

## Запуск

```bash
poetry run python wb_parser.py
```

## Настройка

Все параметры в `config.py`:

| Параметр | Описание |
|----------|----------|
| `SEARCH_QUERY` | Поисковый запрос |
| `DEST` | Регион доставки |
| `MAX_PAGES` | Максимум страниц поиска |
| `DELAY` | Пауза между запросами в секундах |
| `OUTPUT_ALL` | Путь к полному каталогу |
| `OUTPUT_FILTERED` | Путь к отфильтрованному каталогу |
| `FILTER_MIN_RATING` | Минимальный рейтинг для выборки |
| `FILTER_MAX_PRICE` | Максимальная цена |
| `FILTER_COUNTRY` | Страна производства |

## Выходные данные

Создаются два XLSX-файла:

- **Полный каталог** — все найденные товары по категории
- **Отфильтрованный каталог** — товары по фильтру

Колонки: ссылка на товар, артикул, название, цена, описание, ссылки на изображения, характеристики, продавец, ссылка на продавца, размеры, остатки, рейтинг, количество отзывов, страна производства.

## Тесты

```bash
poetry run pytest tests/ -v
```

Покрыта логика: определение basket-сервера по артикулу, построение URL изображений и card.json, фильтрация записей.
